# IPL Data Analysis (2008-2024)

## Overview
Exploratory Data Analysis on IPL match data covering 1090 matches from 2007 to 2024.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Google Colab

## Key Findings
- Mumbai Indians are the most successful team with 140+ wins
- Most teams prefer to field after winning the toss (~57%)
- Winning the toss does not guarantee winning the match
- AB de Villiers is the most awarded Player of the Match

## Dataset
IPL Complete Dataset (2008-2024) — Kaggle
